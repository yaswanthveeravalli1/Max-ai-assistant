## Analysis Rules

### Evidence Standards
- ❌ No fictional assumptions allowed
- ✅ Every conclusion backed by observed chat patterns
- ✅ Multiple examples needed for "Confirmed Pattern" status

### Confidence Classification
| Level | Meaning | Requirement |
|-------|---------|-------------|
| **High** | Definitely observed | 5+ consistent examples |
| **Medium** | Likely pattern | 2-4 examples |
| **Low** | Possible signal | 1 example or indirect evidence |

### Pattern Categories
1. **Confirmed Patterns** — Repeated behavior across multiple days/conversations
2. **Likely Assumptions** — Observed 2-3 times, consistent but limited data
3. **Weak Signals** — Single observation or indirect evidence

### Change Detection
- Personality or emotional changes across different time periods must be detected
- Compare early chat vs recent chat behavior
- Note any significant events that caused behavior changes

### Analysis Metrics
- Texting rhythm (morning/afternoon/evening/night patterns)
- Emotional investment (effort in messages)
- Attachment consistency (stable vs fluctuating)
- Communication energy changes (enthusiasm levels over time)
